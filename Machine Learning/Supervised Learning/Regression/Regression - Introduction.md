- Output is numeric value, learn a numeric function
- Training set: 
	- $$X = \lbrace x^{t,}r^t \rbrace_{t=1}^N  $$
	- $$ r^{t} \epsilon R$$
	- $$ r^{t} = f(x^{t}) + \epsilon$$
- Empirical Error on Training set:
	- $$E(g|X) = \frac{1}{N} \sum_{t=1}^{N}[r^{t}- g(x)^t]^2$$
